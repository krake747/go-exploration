# Bikestore Web Application

This is Go, HTMX and Tailwindcss in action.

## Prerequisites

- [Go](https://go.dev/)
- [Tailwindcss](https://tailwindcss.com/)
- [HTMX](https://htmx.org/)

## Installation

### Tailwindcss

`npm install -D tailwindcss`

### HTMX

`curl -L "https://unpkg.com/htmx.org" > wwwroot/js/htmx.min.js`
